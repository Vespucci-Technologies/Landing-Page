1.1.3

February 7, 2021


What's New ?
  • General improvements to the story editor interface.
  • Bug fixes related to the action-based stories.

1.1.2

January 4, 2021
What's New ?
  • It is now possible to display the filters within the same dimension.
  • Filters are now supported in the StoryFinder.
  • Optimization of the synchronization system, now up twice as fast as before.
  • Fixed a bug related to the non activation of a story.
  • Improved performance of the StoryFinder's probabilistic calculation system.
  • General optimizations.

1.1.1

December 19, 2020
What's New ?
  • macOS BigSur support.
  • All new web dashboard experience.

1.1

November 22, 2020
Segment Integration
  • New platforms support including iOS (Objective-C), Android, React Native, Xamarin.
  • Streamline consumer integration, 1-click integration.
  • Customer can now hold the data.
Improved Experience
  • Dimensions views have all been updated to support Segment’s related improvements.
  • All new settings view, completely re-written.
  • All new usage & billing view.
  • All new onboarding views.
  • UI readability improvements.
Stronger Foundations
  • Improved Sync Engine, more integrated and easier to maintain.
  • New, in-house Location Engine.
  • New server-side customer authentication and quota.

1.0.9

November 7, 2020


What's New ?
  • Fixed a bug that could cause Vespucci to crash when initializing the StoryFinder.
  • Fixed a bug that could cause Vespucci to crash when using the time and technology dimensions.
  • Improvements and upgrades to the import wizard.

1.0.8

October 14, 2020


What's New ?
  • Fixed a bug leading to the creation of duplicates in the Sessions table.
  • Improved readability of tables and database attributes.
  • Improved display of abscissa values for numeric variables.
  • Fixed a bug leading to the erroneous display of the last quantiles in the StoryFinder.
  • Introduction of a brand new parallel calculation system based on Apple Metal for the calculation of the dispersion of qualitative variables.
  • Fixed a bug that caused some classes in the population table not to be initialized.

1.0.7

September 27, 2020


What's New ?
  • Correction of a bug in the display and loading of projects.
  • Fixed a bug that caused Vespucci to crash when applying a geographic filter.
  • Correction of a bug in the calculation of the percentage of explored views.
  • Rationalisation of the StoryFinder code base (reduction of the number of lines of code linked to the probability calculation by 25%).
  • Correction of a bug when applying a filter of the technological dimension.
  • Correction of a bug when viewing data based on language.
  • Fixed a bug affecting the display of filters in dimensional views.
  • Fixed a bug affecting the display of long texts as keywords in the StoryFinder.
  • Rationalisation of the management of keywords in the database.
  • Correction of a bug related to the display of the active story.
  • Fixed a bug related to synchronization status display.
